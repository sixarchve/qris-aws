Architecture Details

Under the hood, the QRIS system uses a compact four-service stack configured for low runtime overhead.

⚡

Go + Gin

High concurrency API routing engine and WebSocket notification hub.

💾

Redis Cache

Cache-aside layer for fast merchant prefetching and transaction state polling.

🗄️

PostgreSQL

Durable source of truth for merchants and payment transactions.

🌐

Nginx Static Runtime

Serves prebuilt React apps and proxies API/WebSocket traffic.
